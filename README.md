# 仕様

変数にスクレイピングするURLを指定する
変数名は
kakakucom_url

URLは
https://kakaku.com/item/J0000039558/spec/#tab

指定したURLだけを参照する

このページから取得する項目は
* 商品名
* メーカー
* 価格
* 画像URL
* スペック
* amazonのURL
